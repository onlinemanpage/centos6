NAME ricci_modclusterd_selinux - Security Enhanced Linux Policy for the ricci_modclusterd processes

DESCRIPTION Security-Enhanced Linux secures the ricci_modclusterd processes via flexible mandatory access control.

The ricci_modclusterd processes execute with the ricci_modclusterd_t SELinux type. You can check if you have these processes running by exe- cuting the ps command with the -Z qualifier.

For example:

ps -eZ | grep ricci_modclusterd_t

ENTRYPOINTS The ricci_modclusterd_t SELinux type can be entered via the "ricci_mod- clusterd_exec_t" file type. The default entrypoint paths for the ricci_modclusterd_t domain are the following:"
