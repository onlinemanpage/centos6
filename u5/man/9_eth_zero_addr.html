ETH_ZERO_ADDR(9) Network device support ETH_ZERO_ADDR(9)

NAME eth_zero_addr - Assign zero address

SYNOPSIS void eth_zero_addr(u8 * addr);

ARGUMENTS addr Pointer to a six-byte array containing the Ethernet address

DESCRIPTION Assign the zero address to the given address array.
