function::user_string_n2

FUNCTION:(3stap) String and data retrieving fun FUNCTION:(3stap)

NAME function::user_string_n2 - Retrieves string of given length from user space

SYNOPSIS user_string_n2:string(addr:long,n:long,err_msg:string)

ARGUMENTS addr the user space address to retrieve the string from

n the maximum length of the string (if not null terminated)

err_msg the error message to return when data isn´t available

DESCRIPTION Returns the C string of a maximum given length from a given user space address. Returns the given error message string on the rare cases when userspace data is not accessible at the given address.
