NAME devicekit_power_selinux - Security Enhanced Linux Policy for the devicekit_power processes

DESCRIPTION Security-Enhanced Linux secures the devicekit_power processes via flex- ible mandatory access control.

The devicekit_power processes execute with the devicekit_power_t SELinux type. You can check if you have these processes running by exe- cuting the ps command with the -Z qualifier.

For example:

ps -eZ | grep devicekit_power_t

ENTRYPOINTS The devicekit_power_t SELinux type can be entered via the "mtrr_device_t,unla- beled_t,proc_type,devicekit_power_exec_t,sysctl_type,filesys- tem_type,file_type" file types. The default entrypoint paths for the devicekit_power_t domain are the following:"

/dev/cpu/mtrr, /usr/libexec/upowerd, /usr/libexec/devkit-power-daemon, all files on the system
