function::read_stopwatch_s

FUNCTION:(3stap) Timestamp Functions FUNCTION:(3stap)

NAME function::read_stopwatch_s - Reads the time in seconds for a stopwatch

SYNOPSIS read_stopwatch_s:long(name:string)

ARGUMENTS name stopwatch name

DESCRIPTION Returns time in seconds for stopwatch name. Creates stopwatch name if it does not currently exist.
