NAME struct_usbdrv_wrap - wrapper for driver-model structure

SYNOPSIS struct usbdrv_wrap { struct device_driver driver; int for_devices; };

MEMBERS driver The driver-model core driver structure.

for_devices Non-zero for device drivers, 0 for interface drivers.
