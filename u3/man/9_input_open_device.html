INPUT_OPEN_DEVICE(9) Input Subsystem INPUT_OPEN_DEVICE(9)

NAME input_open_device - open input device

SYNOPSIS int input_open_device(struct input_handle * handle);

ARGUMENTS handle handle through which device is being accessed

DESCRIPTION This function should be called by input handlers when they want to start receive events from given input device.
