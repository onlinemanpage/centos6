NAME function::set_kernel_short - Writes a short value to kernel memory

SYNOPSIS set_kernel_short(addr:long,val:long)

ARGUMENTS addr The kernel address to write the short to

val The short which is to be written

DESCRIPTION Writes the short value to a given kernel memory address. Reports an error when writing to the given address fails. Requires the use of guru mode (-g).
