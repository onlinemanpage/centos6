function::cpu_clock_ms

FUNCTION:(3stap) Timestamp Functions FUNCTION:(3stap)

NAME function::cpu_clock_ms - Number of milliseconds on the given cpu´s clock

SYNOPSIS cpu_clock_ms:long(cpu:long)

ARGUMENTS cpu Which processor´s clock to read

DESCRIPTION This function returns the number of milliseconds on the given cpu´s clock. This is always monotonic comparing on the same cpu, but may have some drift between cpus (within about a jiffy).
