NAME rhev_agentd_selinux - Security Enhanced Linux Policy for the rhev_agentd processes

DESCRIPTION Security-Enhanced Linux secures the rhev_agentd processes via flexible mandatory access control.

The rhev_agentd processes execute with the rhev_agentd_t SELinux type. You can check if you have these processes running by executing the ps command with the -Z qualifier.

For example:

ps -eZ | grep rhev_agentd_t

ENTRYPOINTS The rhev_agentd_t SELinux type can be entered via the "rhev_agentd_exec_t" file type. The default entrypoint paths for the rhev_agentd_t domain are the following:"

/usr/share/rhev-agent/rhev-agentd.py, /usr/share/rhev-agent/LockAc- tiveSession.py
