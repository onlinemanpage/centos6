NAME ata_port_flush_task - Flush port_task

SYNOPSIS void ata_port_flush_task(struct ata_port * ap);

ARGUMENTS ap The ata_port to flush port_task for

DESCRIPTION After this function completes, port_task is guranteed not to be running or scheduled.

LOCKING Kernel thread context (may sleep)
