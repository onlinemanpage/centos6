NAME virt-who-config - configuration for virt-who

SYNOPISIS /etc/virt-who.d/*

DESCRIPTION Configuration format is ini-like, where group name (in square brackets) is arbitrary name of the configuration.

Only required key is type that has to have one of the allowed virtual- ization backend names, see virt-who(8).

Another options that could be supplied are:

server hostname, IP address or URL of the server that provides virtual- ization information (not applicable for libvirt and vdsm mode).

username username for authentication to the server (not applicable for libvirt and vdsm mode).

password password for authentication to the server (not applicable for libvirt and vdsm mode).

encrypted_password encrypted password that is generated by virt-who-password(8) utility

owner owner for use with Subscription Asset Manager (not applicable for Satellite)

env environment for use with Subscription Asset Manager (not appli- cable for Satellite)

EXAMPLE [test-esx] type=esx server=1.2.3.4 username=admin password=password owner=test env=staging
