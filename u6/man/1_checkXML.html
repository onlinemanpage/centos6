NAME checkXML - An lint tool for DocBook documents.

SYNOPSIS checkXML [filename]

DESCRIPTION checkXML is a tool to check for syntax errors in DocBook files. It can also be used for other DocBook based files, but you should use the less specific xmllint(1) tool if you are not writing or otherwise working on documentation.

SEE ALSO meinproc4(1) xmllint(1)
