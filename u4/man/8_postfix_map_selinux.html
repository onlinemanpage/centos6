NAME postfix_map_selinux - Security Enhanced Linux Policy for the post- fix_map processes

DESCRIPTION Security-Enhanced Linux secures the postfix_map processes via flexible mandatory access control.

The postfix_map processes execute with the postfix_map_t SELinux type. You can check if you have these processes running by executing the ps command with the -Z qualifier.

For example:

ps -eZ | grep postfix_map_t

ENTRYPOINTS The postfix_map_t SELinux type can be entered via the "post- fix_map_exec_t" file type. The default entrypoint paths for the post- fix_map_t domain are the following:"

/usr/sbin/postmap
