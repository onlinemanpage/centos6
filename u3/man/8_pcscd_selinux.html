NAME pcscd_selinux - Security Enhanced Linux Policy for the pcscd processes

DESCRIPTION SELinux Linux secures pcscd (PCSC smart card service) processes via flexible mandatory access control.

FILE CONTEXTS SELinux requires files to have an extended attribute to define the file type.

You can see the context of a file using the -Z option to ls

Policy governs the access confined processes have to these files. SELinux pcscd policy is very flexible allowing users to setup their pcscd processes in as secure a method as possible.

The following file types are defined for pcscd:

pcscd_exec_t

- Set files with the pcscd_exec_t type, if you want to transition an executable to the pcscd_t domain.

pcscd_var_run_t

- Set files with the pcscd_var_run_t type, if you want to store the pcscd files under the /run directory.

Paths: /var/run/pcscd.pid, /var/run/pcscd.comm, /var/run/pcscd.pub, /var/run/pcscd.events(/.*)?

Note: File context can be temporarily modified with the chcon command. If you want to permanently change the file context you need to use the semanage fcontext command. This will modify the SELinux labeling database. You will need to use restorecon to apply the labels.
