function::user_short

FUNCTION:(3stap) String and data retrieving fun FUNCTION:(3stap)

NAME function::user_short - Retrieves a short value stored in user space

SYNOPSIS user_short:long(addr:long)

ARGUMENTS addr the user space address to retrieve the short from

DESCRIPTION Returns the short value from a given user space address. Returns zero when user space data is not accessible.
