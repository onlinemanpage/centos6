NAME perf-test - Runs sanity tests.

SYNOPSIS perf test <options>

DESCRIPTION This command does assorted sanity tests, initially through linked routines but also will look for a directory with more tests in the form of scripts.

OPTIONS -v, --verbose Be more verbose.
