function::read_stopwatch_ns

FUNCTION:(3stap) Timestamp Functions FUNCTION:(3stap)

NAME function::read_stopwatch_ns - Reads the time in nanoseconds for a stopwatch

SYNOPSIS read_stopwatch_ns:long(name:string)

ARGUMENTS name stopwatch name

DESCRIPTION Returns time in nanoseconds for stopwatch name. Creates stopwatch name if it does not currently exist.
