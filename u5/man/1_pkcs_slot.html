NAME pkcs_slot - configuration program for opencryptoki

SYNOPSIS pkcs_slot devicenumber depth

DESCRIPTION This utility is used internal by pkcs11_startup. It writes the config- uration file to disk as /var/lib/opencryptoki/pk_config_data.

SEE ALSO pkcs11_startup(1), opencryptoki(7), pkcsslotd(8), pk_config_data(5).
