NAME varnishlog_selinux - Security Enhanced Linux Policy for the varnishlog processes

DESCRIPTION Security-Enhanced Linux secures the varnishlog processes via flexible mandatory access control.

The varnishlog processes execute with the varnishlog_t SELinux type. You can check if you have these processes running by executing the ps command with the -Z qualifier.

For example:

ps -eZ | grep varnishlog_t

ENTRYPOINTS The varnishlog_t SELinux type can be entered via the "var- nishlog_exec_t" file type. The default entrypoint paths for the var- nishlog_t domain are the following:"

/usr/bin/varnishlog, /usr/bin/varnisncsa
