function::sid

FUNCTION:(3stap) Context Functions FUNCTION:(3stap)

NAME function::sid - Returns the session ID of the current process

SYNOPSIS sid:long()

ARGUMENTS None

DESCRIPTION The session ID of a process is the process group ID of the session leader. Session ID is stored in the signal_struct since Kernel 2.6.0.
