function::exit

FUNCTION:(3stap) Logging Tapset FUNCTION:(3stap)

NAME function::exit - Start shutting down probing script.

SYNOPSIS exit()

ARGUMENTS None

DESCRIPTION This only enqueues a request to start shutting down the script. New probes will not fire (except end probes), but all currently running ones may complete their work.
