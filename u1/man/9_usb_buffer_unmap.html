USB_BUFFER_UNMAP(9) USB Core APIs USB_BUFFER_UNMAP(9)

NAME usb_buffer_unmap - free DMA mapping(s) for an urb

SYNOPSIS void usb_buffer_unmap(struct urb * urb);

ARGUMENTS urb urb whose transfer_buffer will be unmapped

DESCRIPTION Reverses the effect of usb_buffer_map.
