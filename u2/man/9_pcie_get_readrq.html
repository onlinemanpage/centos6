PCIE_GET_READRQ(9) Hardware Interfaces PCIE_GET_READRQ(9)

NAME pcie_get_readrq - get PCI Express read request size

SYNOPSIS int pcie_get_readrq(struct pci_dev * dev);

ARGUMENTS dev PCI device to query

DESCRIPTION Returns maximum memory read request in bytes or appropriate error value.
