NAME subscription-manager-gui - Launches the UI client of the Subscription Manager.

SYNOPSIS subscription-manager-gui

DESCRIPTION subscription-manager-gui opens the local client tool to manage sub- scriptions, entitlements, and installed products on Red Hat Enterprise Linux systems. The same tasks that can be performed by invoking sub- scription-manager can be performed in the GUI, including registering systems with the Red Hat Entitlement Service, subscribing and unsub- scribing to entitlements, and viewing the subscription status of installed products.

The Subscription Manager UI can also be opened by going through the System > Administration menu and selecting the Red Hat Subscription Manager item.

USAGE This command has no options.

ASSOCIATED FILES * /usr/sbin/subscription-manager-gui

* /usr/sbin/subscription-manager

* /etc/rhsm/rhsm.conf

BUGS This client is part of the subscription management system available in Red Hat Enterprise Linux. To file bugs related to this tool, go to https://bugzilla.redhat.com/ and select Red Hat > Red Hat Enterprise Linux # > subscription-manager.
