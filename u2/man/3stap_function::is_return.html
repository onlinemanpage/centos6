FUNCTION::IS_RETURN(3stap) Context Functions FUNCTION::IS_RETURN(3stap)

NAME function::is_return - Whether the current probe context is a return probe

SYNOPSIS is_return:long()

ARGUMENTS None

DESCRIPTION Returns 1 if the current probe context is a return probe, returns 0 otherwise.
