function::read_stopwatch_ms

FUNCTION:(3stap) Timestamp Functions FUNCTION:(3stap)

NAME function::read_stopwatch_ms - Reads the time in milliseconds for a stopwatch

SYNOPSIS read_stopwatch_ms:long(name:string)

ARGUMENTS name stopwatch name

DESCRIPTION Returns time in milliseconds for stopwatch name. Creates stopwatch name if it does not currently exist.
