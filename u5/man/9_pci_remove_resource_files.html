NAME pci_remove_resource_files - cleanup resource files

SYNOPSIS void pci_remove_resource_files(struct pci_dev * pdev);

ARGUMENTS pdev dev to cleanup

DESCRIPTION If we created resource files for pdev, remove them from sysfs and free their resources.
