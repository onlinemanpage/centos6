function::proc_mem_txt

FUNCTION:(3stap) Memory Tapset FUNCTION:(3stap)

NAME function::proc_mem_txt - Program text (code) size in pages

SYNOPSIS proc_mem_txt:long()

ARGUMENTS None

DESCRIPTION Returns the current process text (code) size in pages, or zero when there is no current process or the number of pages couldn´t be retrieved.
