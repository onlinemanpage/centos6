NAME unconfined_notrans_selinux - Security Enhanced Linux Policy for the unconfined_notrans processes

DESCRIPTION Security-Enhanced Linux secures the unconfined_notrans processes via flexible mandatory access control.

The unconfined_notrans processes execute with the unconfined_notrans_t SELinux type. You can check if you have these processes running by exe- cuting the ps command with the -Z qualifier.
