WATERMARK_WAIT(9) Memory Management in Linux WATERMARK_WAIT(9)

NAME watermark_wait - Wait for watermark to go above low

SYNOPSIS long watermark_wait(long timeout);

ARGUMENTS timeout Wait until watermark is reached or this timeout is reached

DESCRIPTION Waits for up to timeout jiffies for watermark on a zone to be reached
