function::task_time_string

FUNCTION:(3stap) Task Time Tapset FUNCTION:(3stap)

NAME function::task_time_string - Human readable string of task time usage

SYNOPSIS task_time_string:string()

ARGUMENTS None

DESCRIPTION Returns a human readable string showing the user and system time the current task has used up to now. For example usr: 0m12.908s, sys: 1m6.851s.
