KZALLOC_NODE(9) Memory Management in Linux KZALLOC_NODE(9)

NAME kzalloc_node - allocate zeroed memory from a particular memory node.

SYNOPSIS void * kzalloc_node(size_t size, gfp_t flags, int node);

ARGUMENTS size how many bytes of memory are required.

flags the type of memory to allocate (see kmalloc).

node memory node from which to allocate
