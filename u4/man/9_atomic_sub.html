NAME atomic_sub - subtract integer from atomic variable

SYNOPSIS void atomic_sub(int i, atomic_t * v);

ARGUMENTS i integer value to subtract

v pointer of type atomic_t

DESCRIPTION Atomically subtracts i from v.
