function::task_utime_tid

FUNCTION:(3stap) Task Time Tapset FUNCTION:(3stap)

NAME function::task_utime_tid - User time of the given task

SYNOPSIS task_utime_tid:long(tid:long)

ARGUMENTS tid Thread id of the given task

DESCRIPTION Returns the user time of the given task in cputime, or zero if the task doesn´t exist. Does not include any time used by other tasks in this process, nor does it include any time of the children of this task.
