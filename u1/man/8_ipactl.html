NAME ipactl - IPA Server Control Interface

SYNOPSIS ipactl command

DESCRIPTION A tool to help an administer control an IPA environment.

IPA glues several discrete services together to work in concert and the order that these services are started and stopped is important. ipactl ensures that they are started and stopped in the correct order.

OPTIONS start Start all of the services that make up IPA

stop Stop all of the services that make up IPA

restart Stop then start all of the services that make up IPA
