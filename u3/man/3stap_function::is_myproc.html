function::is_myproc

FUNCTION:(3stap) Context Functions FUNCTION:(3stap)

NAME function::is_myproc - Determines if the current probe point has occurred in the user´s own process

SYNOPSIS is_myproc:long()

ARGUMENTS None

DESCRIPTION This function returns 1 if the current probe point has occurred in the user´s own process.
