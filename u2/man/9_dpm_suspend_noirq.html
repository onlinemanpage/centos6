DPM_SUSPEND_NOIRQ(9) Device drivers infrastructure DPM_SUSPEND_NOIRQ(9)

NAME dpm_suspend_noirq - Execute late suspend callbacks for non-sysdev devices.

SYNOPSIS int dpm_suspend_noirq(pm_message_t state);

ARGUMENTS state PM transition of the system being carried out.

DESCRIPTION Prevent device drivers from receiving interrupts and call the noirq suspend handlers for all non-sysdev devices.
