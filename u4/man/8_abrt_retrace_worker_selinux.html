NAME abrt_retrace_worker_selinux - Security Enhanced Linux Policy for the abrt_retrace_worker processes

DESCRIPTION Security-Enhanced Linux secures the abrt_retrace_worker processes via flexible mandatory access control.

The abrt_retrace_worker processes execute with the abrt_retrace_worker_t SELinux type. You can check if you have these processes running by executing the ps command with the -Z qualifier.

For example:

ps -eZ | grep abrt_retrace_worker_t

ENTRYPOINTS The abrt_retrace_worker_t SELinux type can be entered via the "abrt_retrace_worker_exec_t" file type. The default entrypoint paths for the abrt_retrace_worker_t domain are the following:"

/usr/bin/abrt-retrace-worker, /usr/bin/retrace-server-worker
