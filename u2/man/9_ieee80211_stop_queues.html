NAME ieee80211_stop_queues - stop all queues

SYNOPSIS void ieee80211_stop_queues(struct ieee80211_hw * hw);

ARGUMENTS hw pointer as obtained from ieee80211_alloc_hw.

DESCRIPTION Drivers should use this function instead of netif_stop_queue.
