function::qs_wait

FUNCTION:(3stap) Queue Statistics Tapset FUNCTION:(3stap)

NAME function::qs_wait - Function to record enqueue requests

SYNOPSIS qs_wait(qname:string)

ARGUMENTS qname the name of the queue requesting enqueue

DESCRIPTION This function records that a new request was enqueued for the given queue name.
