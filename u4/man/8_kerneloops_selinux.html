NAME kerneloops_selinux - Security Enhanced Linux Policy for the kerneloops processes

DESCRIPTION Security-Enhanced Linux secures the kerneloops processes via flexible mandatory access control.

The kerneloops processes execute with the kerneloops_t SELinux type. You can check if you have these processes running by executing the ps command with the -Z qualifier.
