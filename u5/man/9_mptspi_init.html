MPTSPI_INIT(9) Message-based devices MPTSPI_INIT(9)

NAME mptspi_init - Register MPT adapter(s) as SCSI host(s) with SCSI mid-layer.

SYNOPSIS int mptspi_init(void);

ARGUMENTS void no arguments

DESCRIPTION Returns 0 for success, non-zero for failure.
