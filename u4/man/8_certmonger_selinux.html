NAME certmonger_selinux - Security Enhanced Linux Policy for the certmonger processes

DESCRIPTION Security-Enhanced Linux secures the certmonger processes via flexible mandatory access control.

The certmonger processes execute with the certmonger_t SELinux type. You can check if you have these processes running by executing the ps command with the -Z qualifier.

For example:

ps -eZ | grep certmonger_t

ENTRYPOINTS The certmonger_t SELinux type can be entered via the "certmon- ger_exec_t" file type. The default entrypoint paths for the certmon- ger_t domain are the following:"

/usr/sbin/certmonger

Policy governs the access confined processes have to files. SELinux certmonger policy is very flexible allowing users to setup their cert- monger processes in as secure a method as possible.
