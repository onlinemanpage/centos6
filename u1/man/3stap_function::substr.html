NAME function::substr - Returns a substring

SYNOPSIS substr:string(str:string,start:long,length:long)

ARGUMENTS str the string to take a substring from

start starting position of the extracted string (first character is 0)

length length of string to return

DESCRIPTION Returns the substring of the up to the given length starting at the given start position and ending at given stop position.
