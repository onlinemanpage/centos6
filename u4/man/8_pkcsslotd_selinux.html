NAME pkcsslotd_selinux - Security Enhanced Linux Policy for the pkcsslotd processes

DESCRIPTION Security-Enhanced Linux secures the pkcsslotd processes via flexible mandatory access control.

The pkcsslotd processes execute with the pkcsslotd_t SELinux type. You can check if you have these processes running by executing the ps com- mand with the -Z qualifier.

For example:

ps -eZ | grep pkcsslotd_t

/dev/cpu/mtrr, /usr/sbin/pkcsslotd, all files on the system
