NAME ccw_device_is_multipath - determine if device is operating in multipath mode

SYNOPSIS int ccw_device_is_multipath(struct ccw_device * cdev);

ARGUMENTS cdev ccw device

DESCRIPTION Return non-zero if device is operating in multipath mode, zero otherwise.
