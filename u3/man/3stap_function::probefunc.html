function::probefunc

FUNCTION:(3stap) Context Functions FUNCTION:(3stap)

NAME function::probefunc - Return the probe point´s function name, if known

SYNOPSIS probefunc:string()

ARGUMENTS None

DESCRIPTION This function returns the name of the function being probed. It will do this based on the probe point string as returned by pp.

PLEASE NOTE this function is deprecated, please use symname and/or usymname. This function might return a function name based on the current address if the probe point context couldn´t be parsed.
