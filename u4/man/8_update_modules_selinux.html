NAME update_modules_selinux - Security Enhanced Linux Policy for the update_modules processes

DESCRIPTION Security-Enhanced Linux secures the update_modules processes via flexi- ble mandatory access control.

The update_modules processes execute with the update_modules_t SELinux type. You can check if you have these processes running by executing the ps command with the -Z qualifier.

For example:

ps -eZ | grep update_modules_t

ENTRYPOINTS The update_modules_t SELinux type can be entered via the "update_mod- ules_exec_t" file type. The default entrypoint paths for the update_modules_t domain are the following:"

/sbin/modules-update, /sbin/update-modules, /sbin/generate-mod- probe.conf
