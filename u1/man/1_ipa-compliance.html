NAME ipa-compliance - Check entitlement compliance

SYNOPSIS ipa-compliance [OPTION]

DESCRIPTION Verify that the IPA installation is in compliance with the number of client entitlements it has.

Entitlements are managed using the ipa entitle command.

An enrolled host is an machine that has a host keytab in the IPA sys- tem.

The entitlements take the form of x509v3 certificates. The certificates are examined and the quantities summed. This is compared to the number of enrolled hosts to determine compliance.

The command logs to syslog and if run from a tty will log to the termi- nal as well.

The IPA server provides 25 entitlements of its own.

OPTIONS ---debug Enable debugging output in the command

EXIT STATUS 0 if the command was successful

1 if an error occurred

NOTES Entitlements are not checked if the python-rhsm package is not installed.
