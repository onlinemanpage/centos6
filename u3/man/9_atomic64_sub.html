NAME atomic64_sub - subtract the atomic64 variable

SYNOPSIS void atomic64_sub(u64 delta, atomic64_t * ptr);

ARGUMENTS delta integer value to subtract

ptr pointer to type atomic64_t

DESCRIPTION Atomically subtracts delta from ptr.
