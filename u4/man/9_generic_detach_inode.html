NAME generic_detach_inode - remove inode from inode lists

SYNOPSIS int generic_detach_inode(struct inode * inode);

ARGUMENTS inode inode to remove

DESCRIPTION Remove inode from inode lists, write it if it´s dirty. This is just an internal VFS helper exported for hugetlbfs. Do not use!

Returns 1 if inode should be completely destroyed.
