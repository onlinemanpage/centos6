probe::vm.pagefault

PROBE::VM(3stap) Memory Tapset PROBE::VM(3stap)

NAME probe::vm.pagefault - Records that a page fault occurred

SYNOPSIS vm.pagefault

VALUES write_access indicates whether this was a write or read access; 1 indicates a write, while 0 indicates a read

name name of the probe point

address the address of the faulting memory access; i.e. the address that caused the page fault

CONTEXT The process which triggered the fault
