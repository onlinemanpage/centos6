NAME ipa_kpasswd - Proxy Kerberos password change requests

SYNOPSIS ipa_kpasswd

DESCRIPTION Implementation of the kpasswd protocol (RFC 3244).

It is used to proxy password change operations to Directory Server.

ENVIRONMENT VARIABLES KRB5_KTNAME Location of the keytab to be used by ipa_kpasswd

IPA_KPASSWD_DEBUG Enable additional syslog output from ipa_kpasswd. Setting greater than 0 gets basic output. Setting higher than 100 gets more.

EXIT STATUS -1 if an error occurred
