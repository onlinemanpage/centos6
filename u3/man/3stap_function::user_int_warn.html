function::user_int_warn

FUNCTION:(3stap) String and data retrieving fun FUNCTION:(3stap)

NAME function::user_int_warn - Retrieves an int value stored in user space

SYNOPSIS user_int_warn:long(addr:long)

ARGUMENTS addr the user space address to retrieve the int from

DESCRIPTION Returns the int value from a given user space address. Returns zero when user space and warns (but does not abort) about the failure.
