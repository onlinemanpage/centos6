NAME mailman_mail_selinux - Security Enhanced Linux Policy for the mail- man_mail processes

DESCRIPTION Security-Enhanced Linux secures the mailman_mail processes via flexible mandatory access control.

The mailman_mail processes execute with the mailman_mail_t SELinux type. You can check if you have these processes running by executing the ps command with the -Z qualifier.

For example:

ps -eZ | grep mailman_mail_t

ENTRYPOINTS The mailman_mail_t SELinux type can be entered via the "mail- man_mail_exec_t" file type. The default entrypoint paths for the mail- man_mail_t domain are the following:"

/usr/lib(64)?/mailman/mail/mailman, /usr/lib(64)?/mailman/bin/mail- manctl, /usr/lib(64)?/mailman/scripts/mailman
