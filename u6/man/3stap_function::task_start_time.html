NAME function::task_start_time - Start time of the given task

SYNOPSIS task_start_time:long(tid:long)

ARGUMENTS tid Thread id of the given task

DESCRIPTION Returns the start time of the given task in nanoseconds since boot time or 0 if the task does not exist.
