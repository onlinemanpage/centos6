NAME qmail_splogger_selinux - Security Enhanced Linux Policy for the qmail_splogger processes

DESCRIPTION Security-Enhanced Linux secures the qmail_splogger processes via flexi- ble mandatory access control.

The qmail_splogger processes execute with the qmail_splogger_t SELinux type. You can check if you have these processes running by executing the ps command with the -Z qualifier.

For example:

ps -eZ | grep qmail_splogger_t

ENTRYPOINTS The qmail_splogger_t SELinux type can be entered via the "qmail_splog- ger_exec_t" file type. The default entrypoint paths for the qmail_splogger_t domain are the following:"

/var/qmail/bin/splogger
