NAME jabberd_router_selinux - Security Enhanced Linux Policy for the jab- berd_router processes

DESCRIPTION Security-Enhanced Linux secures the jabberd_router processes via flexi- ble mandatory access control.

The jabberd_router processes execute with the jabberd_router_t SELinux type. You can check if you have these processes running by executing the ps command with the -Z qualifier.

For example:

ps -eZ | grep jabberd_router_t
