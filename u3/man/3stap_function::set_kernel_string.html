function::set_kernel_string

FUNCTION:(3stap) String and data writing functi FUNCTION:(3stap)

NAME function::set_kernel_string - Writes a string to kernel memory

SYNOPSIS set_kernel_string(addr:long,val:string)

ARGUMENTS addr The kernel address to write the string to

val The string which is to be written

DESCRIPTION Writes the given string to a given kernel memory address. Reports an error on string copy fault. Requires the use of guru mode (-g).
