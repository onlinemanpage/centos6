NAME regex_milter_selinux - Security Enhanced Linux Policy for the regex_milter processes

DESCRIPTION Security-Enhanced Linux secures the regex_milter processes via flexible mandatory access control.

The regex_milter processes execute with the regex_milter_t SELinux type. You can check if you have these processes running by executing the ps command with the -Z qualifier.

For example:

ps -eZ | grep regex_milter_t

ENTRYPOINTS The regex_milter_t SELinux type can be entered via the "regex_mil- ter_exec_t" file type. The default entrypoint paths for the regex_mil- ter_t domain are the following:"

/usr/sbin/milter-regex
