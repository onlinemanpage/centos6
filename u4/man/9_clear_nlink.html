NAME clear_nlink - directly zero an inode´s link count

SYNOPSIS void clear_nlink(struct inode * inode);

ARGUMENTS inode inode

DESCRIPTION This is a low-level filesystem helper to replace any direct filesystem manipulation of i_nlink. See drop_nlink for why we care about i_nlink hitting zero.
