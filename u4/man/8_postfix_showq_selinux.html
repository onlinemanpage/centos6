NAME postfix_showq_selinux - Security Enhanced Linux Policy for the post- fix_showq processes

DESCRIPTION Security-Enhanced Linux secures the postfix_showq processes via flexi- ble mandatory access control.

The postfix_showq processes execute with the postfix_showq_t SELinux type. You can check if you have these processes running by executing the ps command with the -Z qualifier.

For example:

ps -eZ | grep postfix_showq_t
