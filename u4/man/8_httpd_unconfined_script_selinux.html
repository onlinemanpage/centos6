NAME httpd_unconfined_script_selinux - Security Enhanced Linux Policy for the httpd_unconfined_script processes

DESCRIPTION Security-Enhanced Linux secures the httpd_unconfined_script processes via flexible mandatory access control.

The httpd_unconfined_script processes execute with the httpd_uncon- fined_script_t SELinux type. You can check if you have these processes running by executing the ps command with the -Z qualifier.

For example:

ps -eZ | grep httpd_unconfined_script_t

ENTRYPOINTS The httpd_unconfined_script_t SELinux type can be entered via the "mtrr_device_t,unlabeled_t,proc_type,httpd_uncon- fined_script_exec_t,sysctl_type,filesystem_type,file_type" file types. The default entrypoint paths for the httpd_unconfined_script_t domain are the following:"

/dev/cpu/mtrr, all files on the system
