NAME pmdanetfilter - Linux netfilter IP connection tracking performance metrics domain agent (PMDA)

DESCRIPTION pmdanetfilter is a Performance Metrics Domain Agent (PMDA) which exports metric values from IP connection tracking module in the Linux kernel.

INSTALLATION If you want access to the names and values for the netfilter performance metrics, do the following as root:

# cd $PCP_PMDAS_DIR/netfilter # ./Install

If you want to undo the installation, do the following as root:

# cd $PCP_PMDAS_DIR/netfilter # ./Remove

pmdanetfilter is launched by pmcd(1) and should never be executed directly. The Install and Remove scripts notify pmcd(1) when the agent is installed or removed.

FILES $PCP_PMDAS_DIR/netfilter/Install installation script for the pmdanetfilter agent

$PCP_PMDAS_DIR/netfilter/Remove undo installation script for the pmdanetfilter agent

$PCP_LOG_DIR/pmcd/netfilter.log default log file for error messages from pmdanetfilter

SEE ALSO pmcd(1).
