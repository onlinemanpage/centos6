NAME sge_shepherd_selinux - Security Enhanced Linux Policy for the sge_shep- herd processes

DESCRIPTION Security-Enhanced Linux secures the sge_shepherd processes via flexible mandatory access control.

The sge_shepherd processes execute with the sge_shepherd_t SELinux type. You can check if you have these processes running by executing the ps command with the -Z qualifier.

For example:

ps -eZ | grep sge_shepherd_t

ENTRYPOINTS The sge_shepherd_t SELinux type can be entered via the "mtrr_device_t,unlabeled_t,proc_type,sysctl_type,filesys- tem_type,file_type,sge_shepherd_exec_t" file types. The default entry- point paths for the sge_shepherd_t domain are the following:"

/dev/cpu/mtrr, all files on the system, /usr/bin/sge_shepherd
