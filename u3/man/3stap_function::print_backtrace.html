function::print_backtrace

FUNCTION:(3stap) Context Functions FUNCTION:(3stap)

NAME function::print_backtrace - Print stack back trace

SYNOPSIS print_backtrace()

ARGUMENTS None

DESCRIPTION This function is equivalent to print_stack(backtrace), except that deeper stack nesting may be supported. The function does not return a value.
