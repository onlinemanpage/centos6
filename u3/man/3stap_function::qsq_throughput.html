function::qsq_throughput

FUNCTION:(3stap) Queue Statistics Tapset FUNCTION:(3stap)

NAME function::qsq_throughput - Number of requests served per unit time

SYNOPSIS qsq_throughput:long(qname:string,scale:long)

ARGUMENTS qname queue name

scale scale variable to take account for interval fraction

DESCRIPTION This function returns the average number or requests served per microsecond.
