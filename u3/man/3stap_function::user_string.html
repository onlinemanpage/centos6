function::user_string

FUNCTION:(3stap) String and data retrieving fun FUNCTION:(3stap)

NAME function::user_string - Retrieves string from user space

SYNOPSIS user_string:string(addr:long)

ARGUMENTS addr the user space address to retrieve the string from

DESCRIPTION Returns the null terminated C string from a given user space memory address. Reports <unknown> on the rare cases when userspace data is not accessible.
